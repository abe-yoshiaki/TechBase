## アプリケーションの概要

TwitterとQiitaを掛け合わせたようなサイトです。
url: http://54.92.104.51/

## 機能一覧

- ユーザー新規登録機能(GEM 'devise')
- ユーザーログイン機能(GEM 'devise')
- 記事投稿機能
- 記事一覧表示機能
- 記事詳細表示機能
- 記事編集・削除機能
- コメント投稿機能
- ページネーション機能（非同期）

## 使用技術

- Rails 5.2.3
- MySQL
